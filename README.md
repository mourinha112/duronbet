# DuronBet - Plataforma de Análise de Tênis

Site oficial da DuronBet, plataforma de análise e sinais de tênis.

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide Icons

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

## Deploy na Vercel

1. Faça push do código para um repositório GitHub
2. Conecte o repositório na [Vercel](https://vercel.com)
3. Deploy automático!

Ou use o CLI da Vercel:

```bash
npm i -g vercel
vercel
```
